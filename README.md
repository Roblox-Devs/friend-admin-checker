# friend-admin-checker

Checks if a Roblox user is friends with any admin accounts.

## Installation

To install and run, first simply clone the repository.

```shell
git clone https://github.com/strawbberrys/friend-admin-checker
```

Cd into the folder.

```shell
cd friend-admin-checker
```

Then run the app with ``node``.

```shell
node app
```
